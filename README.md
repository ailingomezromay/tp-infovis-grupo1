### 📝 Descripción del Proyecto
En 2015, 193 países firmaron la **Agenda 2030** con el compromiso de crecer económicamente sin destruir el clima. Este análisis explora si los países están logrando desvincular el crecimiento del Producto Interno Bruto (PIB) de las emisiones de gases de efecto invernadero.

El estudio se centra en tres indicadores clave obtenidos de la API **World Bank Data360**:
1.  **Emisiones de CO₂ per cápita**: Medidas en toneladas métricas.
2.  **PIB per cápita**: Medido en dólares constantes de 2015.
3.  **Intensidad energética**: Uso de energía por unidad de PIB.

### 📊 Visualizaciones Incluidas
El proyecto utiliza una narrativa visual que combina:
- **Gráficos de Líneas**: Evolución temporal de emisiones vs. PIB por país.
- **Gráficos de Dispersión (Scatter Plot)**: Relación entre riqueza y huella de carbono.
- **Mapas de Calor (Heatmaps)**: Intensidad de emisiones a lo largo de las décadas.

### 🛠️ Tecnologías Utilizadas
- **Lenguaje**: Python
- **Análisis de Datos**: `pandas`, `duckdb`
- **Visualización**: `Altair` (Vega-Lite)
- **Frontend**: HTML5, CSS3 (diseño responsive y tipografía avanzada con Google Fonts)
- **Fuentes de datos**: [World Bank Data360 API](https://data360.worldbank.org/en/api)
